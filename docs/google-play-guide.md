## 6. أمان البيانات (Data Safety)

### هل يجمع التطبيق بيانات؟
**نعم**

### ما البيانات التي يجمعها؟

| نوع البيانات | هل يُجمع؟ | هل يُشارك؟ | السبب |
|---|---|---|---|
| الموقع التقريبي | نعم | نعم | مواقيت الصلاة والقبلة |
| الموقع الدقيق | نعم | نعم | اتجاه القبلة |
| معلومات شخصية | لا | لا | - |
| بيانات مالية | لا | لا | - |
| سجل التصفح | لا | لا | - |

### مع من تُشارك البيانات؟
- **Aladhan API** ← لحساب مواقيت الصلاة (يستقبل الإحداثيات فقط)
- **BigDataCloud API** ← لتحديد اسم المدينة (يستقبل الإحداثيات فقط)

### أسئلة إضافية:

| السؤال | الإجابة |
|---|---|
| هل البيانات مشفرة أثناء النقل؟ | نعم (HTTPS) |
| هل يمكن للمستخدم طلب حذف بياناته؟ | لا ينطبق (لا نخزن بيانات) |
| هل التطبيق يتبع سياسة العائلات؟ | لا |
| هل يحتوي إعلانات؟ | لا |
| هل يحتوي مشتريات داخلية؟ | لا |

## 7. الأذونات وتبريرها

Google قد يسأل لماذا تطلب هذه الأذونات:

| الإذن | التبرير |
|---|---|
| ACCESS_FINE_LOCATION | تحديد اتجاه القبلة بدقة وحساب مواقيت الصلاة |
| ACCESS_COARSE_LOCATION | حساب مواقيت الصلاة حسب المدينة |
| POST_NOTIFICATIONS | إرسال تذكيرات يومية بالأذكار والعبادات |
| INTERNET | جلب مواقيت الصلاة والاستماع للقراء |
| RECEIVE_BOOT_COMPLETED | إعادة جدولة التذكيرات بعد إعادة تشغيل الجهاز |
